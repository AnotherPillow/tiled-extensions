# tiled-extension

A collection of Tiled extensions written by AnotherPillow in TypeScript.

## Extensions

- tiled-timelapse
  - Exports screenshots of the current map periodically, to show progress over time.
  - THIS USES A LOT OF STORAGE ON LARGE MAPS
